# Traffic Accident Hotspot Prediction

This project uses Machine Learning to identify accident-prone areas using historical traffic data.

## Features
- Data cleaning & preprocessing
- Exploratory data analysis (EDA)
- ML model training
- Hotspot visualization (heatmap)
- Future risk prediction

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
